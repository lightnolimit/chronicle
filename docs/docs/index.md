# CHRONICLE

Permanent storage for agent and human data on Arweave via Turbo, paid with x402 (USDC on Base).

## What You Can Store

- Notes and markdown
- Images
- JSON data
- Encrypted payloads (client-side)

## How It Works

1. Connect a wallet.
2. Create content in the app.
3. Upload to the permaweb with a small USDC payment.
